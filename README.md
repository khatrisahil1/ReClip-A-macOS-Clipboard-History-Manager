## 🌟 Overview

**ReClip** is a sleek clipboard manager for macOS, designed to boost your productivity by keeping track of everything you copy. Instantly access, search, and manage your clipboard history from the menu bar, with a beautiful and distraction-free interface.

---

## ✨ Features

- 📋 **Clipboard History**: Track & access all recently copied items effortlessly.
- ⚡️ **Lightning-fast Menu Bar App**: Access your clipboard history in seconds.
- ⌨️ **Global Hotkeys**: Assign and customize shortcuts for instant actions.
- 🔎 **Instant Search**: Locate any clipboard entry in real time.
- 🤖 **AI Enhancements** _(Coming Soon)_: Smart summaries, tagging & auto-organization.
- 🖥 **Minimal, Clean UI**: User-first design for a seamless experience.
- 🛠 **Open Source**: Transparent, community-driven development.

## 🚀 Getting Started

### Prerequisites

- macOS
- Xcode with Swift & AppKit support

### Installation

1. **Clone the repository:**
   ```sh
   git clone https://github.com/khatrisahil1/ReClip-A-macOS-Clipboard-History-Manager.git
   cd ReClip-A-macOS-Clipboard-History-Manager
   ```

2. **Install dependencies:**
   ```sh
   pod install
   ```
   _Update this step if you use a different dependency manager._

3. **Open the project:**
   ```sh
   open ReClip.xcodeproj
   ```

4. **Build & Run:**  
   Use Xcode to build and launch the app. You’ll see ReClip in your menu bar!

---

## 🚦 Usage

- Use the menu bar icon to access the clipboard history.
- Set up global hotkeys in the app preferences for even faster access.
- Search your clipboard entries instantly with the search feature.

---

## 🧪 Testing

ReClip uses the `XCTest` framework.

- Run tests in Xcode: <kbd>⌘</kbd> + <kbd>U</kbd>
- Or via command line:
  ```sh
  xcodebuild test -scheme ReClip
  ```
